# Landing Page Background Images

Questa cartella contiene le immagini di background per la landing page.

## File Corrente

- **`landing-bg.jpg`** - Immagine di background principale per la hero section

## Come Aggiungere un'Immagine

1. Scarica un'immagine realistica JPG sul tema "robot collaborativi - automazione business"
2. Ottimizza l'immagine (< 500KB, risoluzione minima 1920x1080)
3. Salva come `landing-bg.jpg` in questa cartella
4. L'immagine verrà caricata automaticamente dalla landing page

## Testare Più Opzioni

Se vuoi testare più immagini:

1. Salva le immagini con nomi diversi:
   - `landing-bg-option-1.jpg`
   - `landing-bg-option-2.jpg`
   - `landing-bg-option-3.jpg`

2. Modifica `BACKGROUND_IMAGE_PATH` in:
   ```
   src/components/landing/landing-page-client.tsx
   ```

Per maggiori informazioni su dove trovare immagini gratuite, vedi:

## Note

- Le immagini devono essere in formato JPG o WebP
- Dimensione consigliata: 1920x1080 o superiore
- Peso consigliato: < 500KB per ottimizzare il caricamento
