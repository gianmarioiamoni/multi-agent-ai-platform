## Suggerimenti per la Scelta

1. **Orientamento**: Preferisci immagini orizzontali (1920x1080 o superiore)
2. **Stile**: Immagini con spazio "vuoto" nella parte superiore o centrale per il testo
3. **Temi da cercare**:
   - Robot collaborativi in ambiente ufficio/industriale
   - Mani umane che lavorano con robot
   - Dashboard e schermi di automazione
   - Team che collabora con tecnologia AI/robotica
   - Spazi di lavoro moderni con automazione

## Come Applicare l'Immagine

1. Scarica l'immagine scelta
2. Salvala in `public/images/landing-bg.jpg` (o un altro nome)
3. Il componente `LandingPageClient` caricherà automaticamente l'immagine
4. Se vuoi provare più immagini, puoi cambiare il nome in `BACKGROUND_IMAGE_PATH`

## Opzioni Multiple

Se vuoi testare più immagini, puoi:

1. Salvare più immagini con nomi diversi:
   - `public/images/landing-bg-option-1.jpg`
   - `public/images/landing-bg-option-2.jpg`
   - `public/images/landing-bg-option-3.jpg`

2. Cambiare `BACKGROUND_IMAGE_PATH` in `landing-page-client.tsx` per testarle

3. O implementare un sistema di selezione dinamica

## Risoluzione Consigliata

- **Minimo**: 1920x1080 (Full HD)
- **Consigliato**: 2560x1440 (2K) o superiore per supporto retina
- **Formato**: JPG (per dimensioni file ottimizzate) o WebP (per prestazioni)

## Note

- Tutte le immagini devono essere ottimizzate per il web (< 500KB se possibile)
- Considera di usare Next.js Image Optimization per migliori prestazioni
- Assicurati che l'immagine abbia un overlay scuro per garantire leggibilità del testo
