# Landing Page Background Image Suggestions

## Theme: Multi-Agent AI Platform

La landing page necessita di un'immagine di background significativa per una piattaforma multi-agent AI. Ecco alcuni suggerimenti di temi e stili:

## Temi Consigliati

### 1. **Network di Agenti Collaborativi**
- Descrizione: Rappresentazione visuale di nodi interconnessi, network graph, agenti che comunicano
- Stile: Astratto, tech, linee di connessione tra nodi
- Colori: Indigo, cyan, blu (coerenti con il tema del sito)
- Esempio: Network graph con nodi luminosi, connessioni dinamiche

### 2. **Automazione e Workflow**
- Descrizione: Flussi di lavoro automatizzati, processi che si eseguono automaticamente
- Stile: Moderno, pulito, con elementi geometrici
- Colori: Gradient indigo/cyan, sfondo scuro
- Esempio: Diagrammi di flusso stilizzati, workflow automation

### 3. **AI e Intelligenza Artificiale**
- Descrizione: Rappresentazione concettuale di AI, neuroni, circuiti
- Stile: Futuristico, tecnologico
- Colori: Electric indigo, vibrant cyan
- Esempio: Brain network, circuit patterns, AI visualization

### 4. **Automazione Business**
- Descrizione: Collaborazione tra agenti per automatizzare task aziendali
- Stile: Professionale, enterprise-grade
- Colori: Scuro con accenti luminosi
- Esempio: Robot collaborativi, automazione aziendale

## Specifiche Tecniche

- **Formato**: JPEG o PNG ad alta risoluzione
- **Dimensioni consigliate**: 1920x1080px o superiore (full HD+)
- **Aspect ratio**: 16:9 o simile
- **Peso**: < 500KB ottimizzato per web
- **Stile**: Abstract, tech-forward, moderno
- **Opacità**: L'immagine sarà applicata con overlay opaco per garantire leggibilità del testo

## Posizionamento

L'immagine sarà utilizzata come:
- Background della hero section
- Con overlay opaco (80-90% opacità) per garantire leggibilità
- Responsive: si adatterà a diverse risoluzioni schermo

## Come Procedere

1. Scegli o genera un'immagine secondo uno dei temi sopra
2. Salva l'immagine in `public/images/landing-bg.jpg` (o formato appropriato)
3. La landing page sarà configurata per utilizzarla automaticamente
